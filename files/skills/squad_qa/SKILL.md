description: Route testing, lint/type quality checks, and low-risk quality remediation to squad_qa.
---

Use this skill for verification-driven work and code-quality hygiene.

Workflow:

1. Select highest-signal checks first.
2. Run targeted validation and classify findings.
3. Apply low-risk fixes when requested.
4. Re-run checks and report before/after.

Return contract:

- Checks run
- Findings by severity
- Fixes applied (if any)
- Remaining blockers
